# 🚀 Space Shooter Mobile: Infinite Boss Edition

Um jogo de arcade rítmico desenvolvido com **JavaScript puro** e **HTML5 Canvas**, otimizado para dispositivos móveis e navegadores desktop.

![Tecnologia](https://img.shields.io/badge/Tecnologia-JavaScript-yellow)

## 🎮 Sobre o Jogo
O **Space Shooter Mobile** é um clássico "shmup" (shoot 'em up) onde o jogador controla uma nave em uma missão infinita pelo espaço. O desafio aumenta progressivamente, culminando em batalhas épicas contra chefes que retornam cada vez mais fortes.

### Funcionalidades Principais:
* **Controles Híbridos**: Jogue no PC (setas e espaço) ou no Celular (arrastar e auto-fire).
* **Boss System**: Chefes cíclicos com IA de movimento lateral e barra de vida.
* **Power-ups**: Sistema de "Tiro Triplo" temporário com efeitos visuais e sonoros.
* **Áudio Adaptativo**: Trilha sonora sintetizada que muda de ritmo e tom quando o Boss aparece.
* **Game Juice**: Efeitos de partículas, tremor de tela (screen shake) e estrelas em parallax.
* **Persistência**: Sistema de High Score utilizando `localStorage`.

## 🕹️ Como Jogar

### No Computador:
* **Setas (Esq/Dir)**: Movimentam a nave.
* **Barra de Espaço**: Atira lasers (segure para tiro contínuo).

### No Celular/Tablet:
* **Deslizar o dedo**: A nave persegue o toque suavemente.
* **Toque na tela**: Ativa o tiro automático.

## 🛠️ Tecnologias Utilizadas
O projeto foi construído "do zero", sem o uso de engines externas, para demonstrar lógica de programação pura:

* **HTML5 Canvas**: Para renderização gráfica de alta performance.
* **JavaScript (ES6+)**: Lógica de jogo, sistema de partículas e detecção de colisão AABB.
* **Web Audio API**: Para geração de sons procedurais e trilha sonora sintetizada.
* **CSS3**: Estilização do container e responsividade mobile.

## 🚀 Como Executar o Projeto

1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Para jogar no celular, você pode hospedar o arquivo no [GitHub Pages](https://pages.github.com/) ou [Vercel](https://vercel.com/).
